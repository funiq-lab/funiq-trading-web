### 1. Get depth market data

`GET` /quote/depth

Retrieve the top-ten depth market data for a specified security, including bid/ask price and quantity for ten levels, basic price info, and trade info.

#### Request headers

Name | Type | Required | Description
---|---|---|---
Content-Type | string | yes | **Fixed value**: "application/json"

#### Query parameters

Name | Type | Required | Description
---|---|---|---
security_code | string | yes | Security code in 6-digit numeric format<br />**Example**: "000001"<br />**Validation rules**:<br />- Length must be 6<br />- Digits only

#### Example request

```bash
curl --location --request GET 'http://localhost:8080/quote/depth?security_code=000001' \
--header 'Content-Type: application/json'
```

#### Response

##### Response fields

Name | Type | Description
---|---|---
code | int | Error code, 0 means success
msg | string | Response message
data | object | Depth market data
security_code | string | Security code
security_name | string | Security name (optional)
market | int | Market type
timestamp | int64 | Data timestamp (microseconds)
update_time | int64 | Update time (HHMMSSMMM format)
trading_phase | int | Trading phase
instrument_status | int | Instrument status
last_price | double | Last price
pre_close_price | double | Previous close price
open_price | double | Open price
high_price | double | High price
low_price | double | Low price
close_price | double | Close price
upper_limit_price | double | Upper limit price
lower_limit_price | double | Lower limit price
avg_price | double | Average price
total_volume | int64 | Total volume
total_value | double | Total turnover
num_trades | int | Number of trades
total_bid_qty | int64 | Total bid quantity
total_ask_qty | int64 | Total ask quantity
weighted_avg_bid_price | double | Weighted average bid price
weighted_avg_ask_price | double | Weighted average ask price
total_bid_number | int | Total number of bid orders
total_ask_number | int | Total number of ask orders
bid_price | array | Top-ten bid prices array
bid_qty | array | Top-ten bid quantities array
ask_price | array | Top-ten ask prices array
ask_qty | array | Top-ten ask quantities array
total_position | int64 | Total position
reference_price | double | Reference price
iopv | double | Real-time NAV
pre_iopv | double | Previous NAV
channel_no | int | Channel number
sequence | int64 | Sequence number
