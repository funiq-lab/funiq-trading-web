## System Introduction

Funiq Trading is a professional quantitative trading system that provides a complete solution for market data ingestion, strategy execution, and trade management. The system follows a modular design and supports multiple market data sources and trading interfaces.

> This program is currently an internal system. More comprehensive security features will be added in the future.

### System Highlights

- Flexible integration: supports multiple market data sources and trading interfaces with quick switching
- Data replay: built-in historical data replay for strategy backtesting and validation
- High-performance design: multi-threaded architecture and optimized data processing to ensure low latency
- Easy to use: provides HTTP API and C API, supports multiple programming languages

## Quick Start

### Configuration

Before starting the system, complete the following configuration:

**Configuration files**
- Configure the required files: `env`, `system_config.json`, `quote_config.json`, `trade_config.json`, `strategy_config.json`
- If you plan to run replays, prepare the replay files; otherwise the local mock data source will be used
- Provide any third-party API credential files required by external providers

### Start the system

After configuration is complete, run the executable:

```bash
# Run the system
./funiq-trading
```

Once the system starts you can operate and query it via the HTTP API.
