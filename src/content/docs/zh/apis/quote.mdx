### 1. 查询深度行情

`GET` /quote/depth

获取指定证券的十档深度行情数据，包括买卖十档价格和数量、基本价格信息、成交信息等。

#### 请求头

名称 | 类型 | 必填 | 描述
---|---|---|---
Content-Type | string | 是 | **固定值**："application/json"

#### 查询参数

名称 | 类型 | 必填 | 描述
---|---|---|---
security_code | string | 是 | 证券代码，6 位数字格式<br />**示例值**："000001"<br />**数据校验规则**：<br />- 长度必须为 6 位<br />- 只能包含数字字符

#### 请求示例

```bash
curl --location --request GET 'http://localhost:8080/quote/depth?security_code=000001' \
--header 'Content-Type: application/json'
```

#### 响应

##### 响应体

名称 | 类型 | 描述
---|---|---
code | int | 错误码，0 表示成功
msg | string | 响应消息
data | object | 深度行情数据
security_code | string | 证券代码
security_name | string | 证券名称（可选）
market | int | 市场类型
timestamp | int64 | 数据时间戳（微秒）
update_time | int64 | 更新时间（HHMMSSMMM格式）
trading_phase | int | 交易阶段
instrument_status | int | 合约状态
last_price | double | 最新价
pre_close_price | double | 昨收价
open_price | double | 开盘价
high_price | double | 最高价
low_price | double | 最低价
close_price | double | 收盘价
upper_limit_price | double | 涨停价
lower_limit_price | double | 跌停价
avg_price | double | 均价
total_volume | int64 | 总成交量
total_value | double | 总成交额
num_trades | int | 成交笔数
total_bid_qty | int64 | 委买总量
total_ask_qty | int64 | 委卖总量
weighted_avg_bid_price | double | 加权平均委买价格
weighted_avg_ask_price | double | 加权平均委卖价格
total_bid_number | int | 委买总笔数
total_ask_number | int | 委卖总笔数
bid_price | array | 十档买入价格数组
bid_qty | array | 十档买入数量数组
ask_price | array | 十档卖出价格数组
ask_qty | array | 十档卖出数量数组
total_position | int64 | 持仓总量
reference_price | double | 参考价
iopv | double | 实时净值
pre_iopv | double | 前日净值
channel_no | int | 频道号
sequence | int64 | 序列号
