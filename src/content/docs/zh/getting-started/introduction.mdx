## 系统简介

Funiq Trading 是一个专业的量化交易系统，为您提供完整的行情接入、策略执行和交易管理解决方案。系统采用模块化设计，支持多种行情数据源和交易接口。

> 此程序目前为内部系统。未来将包含更完善的安全功能。

### 系统特点

- **灵活接入**：支持多种行情源和交易接口，可根据需要快速切换
- **数据回放**：内置历史数据回放功能，方便策略回测和验证
- **高性能设计**：采用多线程架构和优化的数据处理机制，确保低延迟
- **易于使用**：提供 HTTP API 接口及C接口，支持多种编程语言调用


## 功能特性

### ✅ 已支持功能

#### 📡 行情数据

- **多种接入方式**
  - 插件式架构，支持灵活切换行情源
  - 提供三种数据接收模式：共享内存模式（测试中）、轮询模式、回调模式
  
- **多种数据来源**
  - 本地测试数据（Mock）
  - 录制的历史数据
  - CSV 文件导入
  - 实时行情接口
  
- **数据录入与回放**
  - 支持行情数据录入存储
  - 完整的历史数据回放系统，用于策略测试和验证

#### 💹 交易功能

- **插件式交易接口**
  - 灵活的交易接口接入机制
  - 支持多种交易通道
  
- **TWAP 算法**
  - 时间加权平均拆单功能
  - 支持任务创建、查询和取消

#### 🎯 策略管理

- **插件式策略接入**
  - 策略独立部署和管理
  - 支持多策略并行运行

#### 🔧 系统服务

- **网络接口**
  - 完整的 HTTP API 服务
  - 支持行情查询、交易下单、策略控制等操作
  
- **数据管理**
  - 内置嵌入式数据库
  - 数据库版本管理和迁移系统
  
- **日志系统**
  - 分级日志记录
  - 支持文件输出和实时查看
  
- **性能优化**
  - 多线程架构
  - CPU 核心绑定管理
  - 支持高并发场景

- **定时控制**
  - 系统内置定时启停功能
  - 支持外部定时脚本调度

## 快速开始

### 系统配置

在启动系统之前，请先完成以下配置：

**配置文件准备**
   - 按需求配置各配置文件：`env`、`system_config.json`、`quote_config.json`、`trade_config.json` `strategy_config.json`
   - 如果需要回放，则还需要准备回放文件，如果没有正确获取回放文件，则会使用本地mock的数据源
   - 三方API所需要的接口文件

### 启动系统

完成配置后，直接运行系统可执行文件即可：

```bash
# 运行系统
./funiq-trading
```

系统启动后，您可以通过 HTTP API 进行操作和查询。
